# leanirng-webpack2

## Install Webpack
```
# npm 初期化
npm init -y

# version確認
$ npm view webpack
webpack@4.41.6 | MIT | deps: 23 | versions: 639

# install
npm install --save-dev webpack@4.41.6

npm view webpack-cli
webpack-cli@3.3.11 | MIT | deps: 11 | versions: 95

npm install --save-dev webpack-cli@3.3.11

```
